- Python 3.11+
- FastAPI
- Pydantic v2
- PostgreSQL (+ Alembic migrations)

- Celery or RQ workers
- OpenTelemetry (basic tracing/log correlation)

- TypeScript
- Tailwind CSS
- shadcn/ui
- TanStack Query

## AI
- OpenAI API (triage + rule suggestion assistance)

## Infra
- Docker / Docker Compose for MVP
- Optional OpenSearch later (not required for v1)
